填寫退貨資訊

填寫步驟：
  1. 請選擇要退的商品名稱並勾選。
  2. 填寫退貨原因、退貨數量並上傳證明圖片。
  3. 請略過不退貨的商品也不需勾選。

選擇 商品名稱
退貨規範：
  • 退貨商品須為未使用狀態。
  • 無故多次申請退貨或退貨原因資訊不足將依照規範記點懲處。
  • 請在送出申請表後的七天內將商品寄回，否則將不辦理退款。
  • 提交退貨申請即表示您同意我們的退貨條款。
